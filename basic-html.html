How internet works?

Each computer/router has a IP address.

Computers talk to eachother using the TCP/IP protocol.

HTTP (Hyper Text Transfer Protocol) handles web traffic (requests/responses).

HTTPS is for secure connections using SSL certificates.

HTTP is an Application Layer Protocol that runs over TCP and is responsible for handling web traffic between a browser and a web server that stores a web site.

Websites/Webpages are files stored on computers that run a piece of software called a web server.

Hosting companies provides space on a server for your website.

Domanin names can be purchase from registrars and linked to your hosting account/web server.

DNS is used to map IP addresses to domain names.
